Question-1: What are the different ways to select an element in the DOM?

There are many ways to select elements using the DOM , here are some common ones : 1.getElementById, 2.getElementsByClassName, 3.getElementsByTagName, 4.querySelector, 5.querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML : Returns all html content inside element and tags. innerText : Returns all text inside an element. textContent : Used to get the visible & hidden text content of an element.

Question-3: What is event delegation in the DOM?

In event delegation the parent element handles events for child elements,this leverages the "event bubbling" where events triggered on a child element propagate up the DOM tree to its ancestors.

Question-4: What is event bubbling in the DOM?

Event bubbling is a method where it starts from the target element and continues until it reaches the root.

Question-5: How do you create, add, and remove elements using JavaScript?

Using createElement() to create an element, appendChild() to add an element and removeChild() to remove an element.
